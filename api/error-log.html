Error Log
Every correction to a published scorecard, documented in full.
When a scorecard is found to contain an error — a promise misclassified, evidence overlooked, or a score that doesn't match the documented record — the correction is made and logged here with the original score, the corrected score, the evidence that prompted the change, and the date. Corrections submitted by officials, staff, or the public go through the same review process. To report an error, use the "Report an error" link on any scorecard, or visit the About page for more on the dispute process. Every submission we receive is tracked publicly through review on the Submission Tracker.
